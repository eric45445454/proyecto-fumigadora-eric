01 — El Proyecto

¿Qué es este sistema?

Este proyecto nace como una solución práctica y funcional al problema del control operativo en sistemas de fumigación de alta demanda.

La base del sistema fue concebida y desarrollada como una unidad mecánica motorizada, robusta y eficiente. A este sistema consolidado se le incorporó una capa tecnológica orientada al monitoreo inteligente, permitiendo supervisar una zona térmica clave del proceso sin la necesidad de intervención manual constante.

La idea central es garantizar que el sistema opere dentro de un rango térmico óptimo, mejorando la eficiencia del proceso de fumigación y reduciendo el margen de error operativo.

El problema

La falta de monitoreo térmico en tiempo real en sistemas de fumigación genera ineficiencias operativas y mayor dependencia del criterio humano.

La solución

Integración de sensórica y control digital al sistema mecánico existente, creando un sistema híbrido con mayor precisión y autonomía.

La innovación

La fusión entre un sistema mecánico consolidado y una capa IoT accesible desde la web representa un enfoque moderno, escalable y práctico.

02 — Funcionamiento

El flujo de operación

El sistema opera mediante una secuencia lógica que integra hardware y software de forma coordinada.

01

Monitoreo térmico

El sensor captura la temperatura de una zona clave dentro del sistema, donde la condición térmica determina la eficiencia del proceso.

02

Lectura del ESP32

El microcontrolador ESP32 recibe y procesa los datos del sensor en tiempo real, comparándolos continuamente con el umbral configurado.

03

Visualización web

Los datos son enviados a una interfaz web que muestra la temperatura actual, el estado del sistema y la configuración de umbral.

04

Umbral de activación

El operador define el umbral térmico óptimo desde la interfaz. El sistema evalúa constantemente si la condición está dentro del rango deseado.

05

Activación del sistema

Cuando se alcanza la condición térmica adecuada, el sistema entra en operación de forma controlada, garantizando eficiencia en el proceso.

06

Control optimizado

El resultado es un sistema más preciso, menos dependiente de la supervisión manual, con trazabilidad de datos y operación coherente.

05 — Mi Aporte

La capa tecnológica

El sistema base fue desarrollado por mi papá, quien diseñó y construyó la estructura mecánica funcional que da vida a este proyecto.

Mi contribución fue complementar esa base sólida con una capa digital e inteligente: integrar el ESP32, implementar la lectura del sensor de temperatura, desarrollar la interfaz web de monitoreo y configurar la lógica de activación por umbral térmico.

El resultado es un sistema completo que combina experiencia mecánica y pensamiento digital — un trabajo compartido donde cada parte suma.

Interfaz web de monitoreo

08 — Proceso

Evolución del proyecto

Fase 01

Idea inicial

Identificación de la necesidad de un sistema de fumigación más eficiente y controlado para uso práctico y real.

Fase 02

Desarrollo del sistema base

Diseño y construcción de la estructura mecánica motorizada: el núcleo funcional del proyecto, desarrollado por mi papá.

Fase 03

Incorporación del monitoreo

Análisis de la zona clave del sistema y definición de la variable a monitorear: temperatura de operación.

Fase 04

Integración del ESP32

Selección del microcontrolador, conexión del sensor de temperatura y configuración del firmware de lectura de datos.

Fase 05

Desarrollo de la interfaz web

Diseño y programación del panel de monitoreo accesible desde el navegador, con visualización de temperatura y estado del sistema.

Resultado

Sistema mecatrónico funcional

Integración completa de la parte mecánica y la capa digital, resultando en un sistema cohesivo, controlado y presentable como proyecto de innovación.
